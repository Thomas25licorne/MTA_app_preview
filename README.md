# MTA_app_preview
The following is a preview of what the device will display on your screen. It allows you to know the style it will be in.

Step 1: Ensure the following are downloaded:

pip install flask requests google-cloud-transit protobuf pandas

Step 2: Execute build_database.py

Step 3: Execute app.py and click on the link in the terminal. It should open a web page.
