# NYC Subway

**-Have a virtual environment created and enabled when executing files**

**-download the following packages: pip install flask pandas requests gtfs-realtime-bindings**


1. Download the whole folder.
2. Go to https://www.mta.info/developers and download the GTFS file named "regular GTFS"
3. Create a folder named "data"
4. In the folder "data", add the files "stops.txt", "stop_times.txt", "routes.txt", and "trips.txt" from the GTFS download
6. Execute the file "build_database.py"
7. Execute "app.py"
